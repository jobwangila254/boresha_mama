# Boresha-Mama CHV Training Guide

## Training Session Outline (Half Day)

### Session 1: Introduction (30 min)
- What is Boresha-Mama?
- Why digital maternal health records?
- Overview of app features
- Privacy and confidentiality

### Session 2: App Basics (45 min)
- Installing the app
- Logging in
- Navigating the interface
- **Practice:** Login and explore

### Session 3: Pregnancy Registration (45 min)
- When to register a pregnancy
- Information needed:
  - Mother's name and phone
  - LMP date (Last Menstrual Period)
  - Previous pregnancies (gravida/parity)
  - Risk factors
- **Practice:** Register a test pregnancy

### Session 4: Home Visits (60 min)
- Recording vitals:
  - Weight, BP, temperature
  - Fundal height, fetal heart rate
  - Hemoglobin, pulse
- Recognizing danger signs:
  - Severe headache, blurred vision
  - Vaginal bleeding, abdominal pain
  - Convulsions, fever, swelling
  - Reduced fetal movements
- **Practice:** Record a full home visit

### Session 5: Offline Mode (30 min)
- How offline storage works
- Syncing when online
- Checking sync status
- **Practice:** Enable airplane mode, record visit, sync later

### Session 6: Referrals (30 min)
- When to refer a mother
- How to create a referral
- Tracking referral status
- **Practice:** Create a test referral

### Session 7: Q&A and Assessment (30 min)
- Open questions
- Practical test:
  1. Register a pregnancy
  2. Record vital signs
  3. Create a referral
  4. Demonstrate offline sync

## Key Messages for CHVs
1. **Data saves lives** - Accurate records help detect problems early
2. **Privacy matters** - Mother's health data is confidential
3. **Sync regularly** - Keep data flowing to clinics
4. **Refer quickly** - Don't delay emergency referrals
5. **Follow up** - Track referred mothers and visit schedules
