# Boresha-Mama CHV App - User Manual

**Language:** English / Kiswahili

## Getting Started

1. **Login** with your CHV credentials (provided by county health department)
2. **Dashboard** shows assigned mothers and sync status

## Key Features

### Offline-First Design
- Record home visits even without internet
- Data syncs automatically when connection is available
- Check sync status on the home screen banner
- Tap banner to manually sync pending data

### Register Pregnancy
1. Tap "Register Pregnancy"
2. Enter mother's phone number and name
3. Record LMP date (auto-calculates EDD)
4. Select risk factors if any
5. Submit (works offline too)

### Record Home Visit
1. Tap "Record Home Visit"
2. Select the mother from your list
3. Fill vital signs:
   - Weight (kg)
   - Blood pressure (systolic/diastolic)
   - Temperature (°C)
   - Pulse rate
   - Hemoglobin
   - Fundal height (cm)
   - Fetal heart rate
4. Check any danger signs present
5. Add notes and save

### Risk Assessment
- BP ≥ 140/90 → High risk, consider referral
- Danger signs present → High risk alert
- System auto-calculates risk level

### Create Referral
1. Select the mother
2. Choose receiving facility
3. Enter referral reason
4. Set priority (normal/urgent/emergency)
5. Track referral status from dashboard

### Follow-up Tracking
- View assigned mothers list
- See risk levels at a glance (color-coded)
- Track visit history per mother
- View pregnancy timeline

## Sync Status Indicators
- **Green:** Online - data syncing
- **Red:** Offline - data saved locally
- **Number badge:** Pending items to sync

## Troubleshooting

**Can't login?** Contact county admin to verify your account
**Sync not working?** Check internet connection; tap sync banner
**Wrong data entered?** Contact facility staff to correct records
