# Boresha-Mama County Portal - User Manual

**Access:** https://county.boreshamama.go.ke

**Role:** County Health Administrators

## Dashboard
Real-time county-wide maternal health KPIs:
- Total registered mothers
- Active pregnancies
- High-risk cases
- Active CHVs and facilities
- ANC coverage percentage
- Monthly trends (bar chart)
- Risk distribution (pie chart)

## Analytics
- **Trend charts:** Pregnancies, referrals, home visits over time
- **High-risk tracking:** Monitor high-risk pregnancy trends
- **Period filter:** Monthly or quarterly views
- Data-driven planning and resource allocation

## Facilities Management
- View all facilities in Trans-Nzoia
- Filter by type, ward, constituency
- Monitor facility status (active/inactive)
- Contact information for each facility

## User Management
- Manage CHV accounts
- Manage facility staff accounts
- View registered mothers
- Control account activation/deactivation

## Reports & Export
- Generate aggregated reports:
  - Pregnancy records (with risk levels)
  - Referral tracking data
  - CHV performance metrics
- Export as CSV for further analysis
- Data supports SDG 3 and UHC reporting

## Security
- Role-based access control
- All data encrypted in transit and at rest
- Audit logging for all actions
- Compliant with Kenya Data Protection Act 2019

## Best Practices
- Review KPIs weekly for early intervention
- Use trend data for resource planning
- Monitor CHV performance for training needs
- Ensure data privacy at all times
